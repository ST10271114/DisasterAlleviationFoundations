This is a web application built with ASP.NET Core MVC and Entity Framework Core for managing disaster relief activities.
It allows users to:

Register and log in securely using ASP.NET Identity.

Report and view Incidents.

Make and track Donations.

Sign up for Volunteer Tasks.

View and update their Profile.

The system is connected to an Azure SQL Database where all users, donations, volunteers, and incidents are stored.

**How to Use**

Register a new user account.

Login to access Incidents, Donations, and Volunteer Tasks.

Logged-in users can add, view, and manage their records.
